# Hardware
## Requirements
* Ra-02 LoRa module with SX1278 chip
* 433MHz antenna
* STM32 microcontroller

## Wiring
The wire connections are like following:
| STM pin | Module pin |
| ------- | ---------- |
|  PA7   |    MISO    |
|  PA6   |    MOSI    |
|  PA5   |    CLK    |
|  PB0   |    NSS    |
|  3.3v   |    3.3v    |
|  GND   |    GND    |
